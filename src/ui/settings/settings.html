Settings
Appearance
Application The following settings only affect the actual tool and not the appearance of the 4Blocker or the Report.
Theme Controls the color theme for the entire application
Font size Controls the font size (in pixels) throughout the entire application.
Default Zoom Controls the default zoom level for the application (Described as a percentage).
4Blocker All settings regarding the appearance of the 4Blocker section of the tool.
Report All settings regarding the appearance of the Report section of the tool.
Behavior
General The following settings affect the behavior of both the 4Blocker and reporting sections of the tool.
Progress threshold for accomplishments Minimum threshold value for which an activity is considered an accomplishment.
Behind task count for Behind (Project) How many tasks "Behind" for the project to reflect a status of "Behind".
Late task count for Late (Project) How many tasks "Late" for the project to reflect a status of "Late".
Week count for Behind (Task) How many weeks behind plan for a task to reflect a status of "Behind" (For Weekly Projects Only)
Day count for Behind (Task) How many business days behind plan for a task to reflect a status of "Behind" (For Daily and Scrum Projects Only)
Week count for Late (Task) How many weeks behind plan for a task to reflect a status of "Out of Track" (For Weekly Projects Only)
Day count for Late (Task) How many business days behind plan for a task to reflect a status of "Out of Track" (For Daily and Scrum Projects Only)
4Blocker The following settings only affect the behavior of the 4Blocker part of the tool.
Reporting weeks Controls the number of weeks that an accomplishment is shown in the Blocker. (For Weekly Projects Only)
Reporting days Controls the number of business days that an accomplishment is shown in the Blocker. (For Daily and Scrum Projects Only)
Report The following settings only affect the Report section of the tool.
Reporting weeks Controls the number of weeks that an accomplishment is shown in the Report. (For Weekly Projects Only)
Reporting days Controls the number of business days that an accomplishment is shown in the Report. (For Daily and Scrum Projects Only)
